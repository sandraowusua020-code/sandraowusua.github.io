## Deployment

### GitHub Pages
1. Create a GitHub repository named `sandraowusua.github.io`
2. Upload all files and folders
3. Go to **Settings → Pages**, select **main branch**, save
4. Your portfolio is live at: `https://sandraowusua.github.io`

### Optional Custom Domain
- Add a `CNAME` file with `sandraowusua.myportfolio.site`
- Update DNS records to point to GitHub Pages IP addresses
